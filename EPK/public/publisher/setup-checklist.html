First-run setup checklist

Use this before treating the hosted publisher as private or publishing live EPK data.

Back to publisher

Privacy and deployment

  • Protect /publisher/* with Cloudflare Access or equivalent before relying on hosted publisher privacy.
  • Keep the deploy root at EPK/public, or set Cloudflare project root to EPK with output public.
  • Everything in the deployed public root is publicly servable unless platform protection is configured.
  • Do not place private media, private notes, or credentials in the public deploy root.

Publishing safety

  • Validate the JSON before publishing live data.
  • Download or export JSON before publishing significant changes.
  • Use a publishing credential only for the current session.
  • Publish snapshots for fixed shareable versions at /published/<id>/.
  • Public pages must not link back to publisher/admin tools.

Local first-run commands

python3 -m http.server 8000

Then open http://localhost:8000/EPK/public/, http://localhost:8000/EPK/public/publisher/, and http://localhost:8000/EPK/admin/admin.html.
